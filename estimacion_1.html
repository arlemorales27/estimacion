Parte 1: Conceptos Fundamentales

1.1 ¿Qué es Estimar en Software y por qué es Crucial?

Discusión inicial: Pregunta a los estudiantes: "¿Por qué creen que es importante estimar cuánto tiempo y dinero costará un proyecto de software?"

Explicación:

  • Para el negocio: Permite tomar decisiones informadas, asegurar presupuestos y planificar estrategias de lanzamiento.
  • Para el equipo de desarrollo: Ayuda a gestionar expectativas, planificar el trabajo y evitar el agotamiento.
  • Para el cliente: Proporciona transparencia y confianza en la inversión.

1.2 Precisión vs. Exactitud: El Cono de la Incertidumbre

Concepto clave: La estimación no es una ciencia exacta. Al inicio, las estimaciones tienen un rango amplio (60%-160%), pero se reducen con más información.

Imagen del Cono de la Incertidumbre (Fuente: Wikimedia Commons)

1.3 Tipos de Costos en un Proyecto de Software

Definición de Costos Directos: Son los gastos atribuibles directamente al desarrollo del proyecto de software, relacionados con los recursos específicos utilizados en la creación del producto.

  • Salarios y Honorarios: Costo del tiempo del equipo (desarrolladores, diseñadores, testers).
  • Software y Licencias: Herramientas, IDEs, sistemas operativos necesarios para el desarrollo.
  • Hardware y Equipos: Computadores, servidores dedicados al proyecto.

Definición de Costos Indirectos (Overhead): Son los gastos generales que no se asignan directamente al proyecto, pero que son necesarios para mantener el entorno operativo de la organización.

  • Infraestructura: Oficinas, internet, electricidad compartida por múltiples proyectos.
  • Administrativos: Personal de RRHH, contabilidad, gerencia que apoya varios proyectos.
  • Beneficios: Prestaciones sociales, seguros médicos para el personal.

Descarga la plantilla:

presupuesto_proyecto_v1.xlsx

Paso 2: Escribir el Script de Python

Crea un archivo semana1_calculadora.py con el siguiente código:

# semana1_calculadora.py
import pandas as pd

# Nombre del archivo de Excel
archivo_excel = 'presupuesto_proyecto_v1.xlsx'
nombre_hoja = 'CostosDirectos'

print(f"--- Leyendo el archivo: {archivo_excel} ---")

# Leer el archivo Excel usando pandas
try:
    df_costos = pd.read_excel(archivo_excel, sheet_name=nombre_hoja)

    # Mostrar los datos leídos
    print("\nDatos leídos desde Excel:")
    print(df_costos)

    # Calcular la columna 'Costo Total (COP)'
    df_costos['Costo Total (COP)'] = df_costos['Horas Estimadas'] * df_costos['Tarifa por Hora (COP)']

    # Calcular el costo total del proyecto
    costo_total_proyecto = df_costos['Costo Total (COP)'].sum()

    # Mostrar los resultados
    print("\n--- Presupuesto Calculado ---")
    print("\nDetalle de costos por tarea:")
    print(df_costos)

    print("\n-----------------------------------------------------")
    print(f"Costo Total Directo del Proyecto: ${costo_total_proyecto:,.2f} COP")
    print("-----------------------------------------------------")

except FileNotFoundError:
    print(f"\n¡ERROR! No se encontró el archivo '{archivo_excel}'.")
    print("Asegúrate de que el archivo esté en la misma carpeta que el script.")

Paso 3: Ejecutar el Script

Con el entorno virtual activado, ejecuta:

python semana1_calculadora.py
